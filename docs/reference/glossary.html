Glossary: Physical AI & Humanoid Robotics

A​

AI (Artificial Intelligence): The simulation of human intelligence processes by machines, especially computer systems.

Autonomous Navigation: The ability of a robot to move through an environment without human guidance.

B​

Behavior Trees: A mathematical model of plan execution used in robotics and AI.

C​

Computer Vision: A field of artificial intelligence that trains computers to interpret and understand the visual world.

D​

Digital Twin: A virtual representation of a physical object or system that mirrors its real-world counterpart.

E​

Embodied AI: Artificial intelligence systems that interact with the physical world through robotic bodies.

G​

Gazebo: A 3D multiple robot simulator providing high-fidelity physics simulation.

H​

Humanoid Robot: A robot with a human-like body structure, typically having a head, torso, arms, and legs.

I​

Isaac Sim: NVIDIA's robotics simulation application for developing and testing AI-powered robots.

L​

Large Language Model (LLM): A language model that uses deep learning to understand and generate human-like text.

N​

Navigation: The process of planning and executing paths for mobile robots in an environment.

R​

ROS (Robot Operating System): A flexible framework for writing robot software.

ROS 2: The second generation Robot Operating System providing communication infrastructure for robotic applications.

S​

SLAM (Simultaneous Localization and Mapping): The computational problem of constructing or updating a map of an unknown environment while simultaneously keeping track of an agent's location within it.

U​

URDF (Unified Robot Description Format): An XML format for representing robot models in ROS.

V​

Vision-Language-Action (VLA): Systems that integrate visual perception, natural language understanding, and action execution for robotic applications.
